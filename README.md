# All Shade Jets | Luxury Private Aviation 🛩️

An immersive, Awwwards-inspired luxury private aviation platform built to deliver a premium, high-performance web experience. Combining advanced scroll animations, 3D visual effects, and fluid typography, this application redefines digital luxury travel interfaces.

## 🌐 Live Demo

**[View Live Website →](https://clone-jesko-jets.vercel.app)**

---

## 🚀 Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/), Vanilla CSS
- **Animations:** [GSAP](https://gsap.com/) (ScrollTrigger, SplitText), [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics:** [Three.js](https://threejs.org/), [@react-three/fiber](https://r3f.docs.pmnd.rs/), [@react-three/drei](https://github.com/pmndrs/drei)
- **Smooth Scrolling:** [Lenis](https://lenis.studiofreight.com/)

---

## ✨ Key Features

- **Cinematic Scroll Animations:** Built with GSAP ScrollTrigger to orchestrate complex timeline animations as users navigate the page.
- **3D Parallax Environments:** Utilizes Three.js and custom parallax logic for immersive sky backgrounds and cloud depth effects.
- **Dynamic Content Sections:** Framer Motion powers smooth transitions for the "Features" and "Flight" sections.
- **Lenis Smooth Scrolling:** Provides a buttery-smooth scrolling experience across devices, overriding native scroll constraints.
- **Modern Responsive Design:** Custom mobile views and fully optimized UI elements ensure the experience is flawless on any screen size.
- **Performance Optimized:** Uses Next.js image optimization (`next/image`) and efficient rendering strategies for media-heavy components.

---

## 📂 Project Structure

- `app/components/HeroSection.jsx` - Dynamic hero area with text masking and scaling image animations.
- `app/components/SkyBackground.jsx` - Multi-layered 3D/parallax sky and cloud environment.
- `app/components/FlightSection.jsx` - Scroll-linked aircraft blueprint animation and spec showcases.
- `app/components/FeaturesSection.jsx` - Interactive service features accordion with image reveal transitions.
- `app/components/SmoothScroll.jsx` - Custom Lenis scroll integration.

## ⚙️ Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mohamedfaisal.dev/Clone-Jesko-jets.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)
